# Стратегия продвижения — Web Toolkit

## 1. Контент (SEO-страницы)

Добавить страницы-гайды, которые Google проиндексирует и которые будут приводить трафик к инструментам.

| Страница | Целевой запрос | Содержание |
|----------|---------------|------------|
| `/blog/how-to-format-json` | "how to format json online" | Гайд по форматированию JSON + ссылка на редактор |
| `/blog/cron-syntax-guide` | "cron expression explained" | Таблица синтаксиса cron + парсер |
| `/blog/jwt-token-explained` | "what is jwt token" | Как работает JWT + декодер |
| `/blog/base64-encoding-guide` | "base64 encoding explained" | Зачем нужен Base64 + инструмент |
| `/blog/regex-cheat-sheet` | "regex cheat sheet" | Шпаргалка по regex + тестер |
| `/blog/json-schema-guide` | "json schema tutorial" | Введение в JSON Schema + генератор |
| `/blog/yaml-vs-json` | "yaml vs json difference" | Сравнительная статья + конвертер |
| `/blog/uuid-guide` | "what is uuid v4" | UUID простым языком + генератор |

Каждый гайд приводит органический трафик и направляет пользователей к соответствующему инструменту.

**Формат:** статические HTML-страницы (не SPA) для лучшей индексации. Включить внутренние ссылки на инструменты. Добавить хлебные крошки и структурированные данные.

**Русские версии:** дублировать каждый гайд на русском (`/ru/blog/...`) для трафика из Яндекса.

## 3. Техническое SEO

- [ ] **Купить домен** (~$10/год) — например `devtoolbox.app`, `devtools.online`. Самый сильный буст для SEO. Субдомен GitHub Pages имеет очень низкий уровень доверия.
- [ ] **Sitemap.xml** — сгенерировать и отправить в Google Search Console и Яндекс.Вебмастер
- [ ] **robots.txt** — убедиться, что все страницы доступны для краулеров
- [ ] **Скорость загрузки** — уже хорошая (SPA), но проверить через Lighthouse
- [ ] **Мобильный UX** — проверить, что все инструменты работают на мобильных

## 5. Порядок приоритетов

1. **Купить домен** — максимальный эффект, минимум усилий
2. **Статья на Dev.to** — легко написать, лояльное сообщество
3. **Статья на Habr** — захват русского рынка
4. **Пост на Reddit** — быстрый всплеск трафика
5. **PR в Awesome-списки** — постоянные бэклинки
6. **Страницы-гайды (блог)** — долгосрочный органический рост
7. **Запуск на Product Hunt** — когда всё остальное отполировано
8. **Hacker News** — непредсказуемо, но высокий потолок
