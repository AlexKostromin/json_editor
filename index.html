DevTools Online — Free Developer Tools

DevTools Online is a collection of 24 free online tools for web developers, backend engineers, DevOps and QA specialists. All tools run directly in your browser, no data is sent to any server. Can be installed as a PWA app for offline use.

Online JSON Editor

Free JSON Editor Online with syntax highlighting, formatting, validation, and tree view. Supports tree, text, and table modes for working with JSON data of any size. Upload and download files, paste from clipboard.

JWT Decoder — Token Decoding

Online JWT decoder: paste a JSON Web Token and instantly get decoded header and payload. Automatic conversion of Unix timestamp fields (exp, iat, nbf) to readable dates. Secure — the token never leaves your browser. Works with Auth0, Firebase, Keycloak and any other JWTs.

Base64 Encode / Decode

Online Base64 encoding and decoding. Full Unicode support. Convert text to Base64 and back instantly in your browser. Use for APIs, email attachments, and configuration files.

URL Encode / Decode

Online URL encoding and decoding. Convert special characters and spaces to URL-safe format and back. Essential for query parameters, API requests, and web links.

Unix Timestamp Converter

Convert Unix Timestamp to human-readable date and back. Supports seconds and milliseconds, displays in UTC and local time. Useful for analyzing logs, databases, and API responses.

UUID v4 Generator

Online random UUID v4 generator. Create unique identifiers for databases, APIs, and testing. Generates 5 UUIDs at once using cryptographically secure crypto.randomUUID().

YAML ↔ JSON Converter

Online YAML to JSON and back conversion. Instantly convert configuration files for Kubernetes, Docker Compose, Ansible, GitHub Actions between formats. All YAML data types supported.

JSON Schema Generator

Automatic JSON Schema (draft 2020-12) generation from JSON data. Detects types, formats (date-time, email, uri) and required fields. Use for API validation, OpenAPI specs, and configurations.

Hash Generator — SHA-256, SHA-1, SHA-512

Online SHA-256, SHA-1, SHA-384, SHA-512 hash generation. Text hashing via Web Crypto API right in your browser. Secure — data never leaves your computer. Suitable for data integrity verification and checksums.

Regex Tester — Regular Expression Testing

Online regex tester with g, i, m flag support. Shows all matches, capture groups, and positions. Essential tool for debugging regex in JavaScript, Python, Go, and other languages.

Color Converter — HEX, RGB, HSL

Online color converter between HEX, RGB, and HSL formats. Instant conversion of color values for CSS, Tailwind, Material Design. Supports 3- and 6-digit HEX codes.

JSON Diff — JSON Comparison

Online comparison of two JSON objects. Deep recursive comparison counting added, removed, and changed fields. Shows the path to each change. Useful for debugging APIs and comparing configurations.

Mock Data Generator — Fake Data

Fake data generator for testing. Create users with names, emails, phones, cities, companies, and other fields. Up to 100 records at once. Perfect for seed data, tests, and prototyping.

Cron Expression Parser

Online cron expression parser. Convert cron syntax like */5 * * * * into human-readable descriptions. Shows next 5 scheduled run times. Supports standard 5-field cron format. Essential for debugging crontab, Kubernetes CronJobs, and CI/CD schedules.

Password Generator — Secure Random Passwords

Cryptographically secure password generator using Web Crypto API. Customize length (4-128 chars), character sets (uppercase, lowercase, numbers, symbols). Generate up to 20 passwords at once with entropy calculation and strength indicator.

Lorem Ipsum Generator — Placeholder Text

Generate Lorem Ipsum placeholder text by words, sentences, or paragraphs. Perfect for mockups, wireframes, and design prototyping. Includes character and word count.

Number Base Converter — HEX, DEC, OCT, BIN

Online number base converter. Convert between decimal, hexadecimal, octal, and binary number systems. Supports 0x, 0b, 0o prefixes. Essential for low-level programming, networking, and embedded systems.

Case Converter — camelCase, snake_case, kebab-case

Online case converter. Transform text between camelCase, PascalCase, snake_case, SCREAMING_SNAKE_CASE, kebab-case, Title Case, dot.case and more. Essential for code refactoring and naming conventions.

HTML Entity Encoder / Decoder

Online HTML entity encoding and decoding. Convert special characters to HTML entities (&amp; &lt; &gt; &quot;) and back. Supports named entities and numeric character references. Essential for web development and XSS prevention.

String Escape / Unescape

Online string escape and unescape tool. Escape strings for JSON, JavaScript, HTML, URL, and CSV formats. Unescape encoded strings back to original text. Essential for debugging, API development, and data processing.

Query String ↔ JSON Converter

Online converter between URL query strings and JSON objects. Parse ?foo=1&bar=2 into structured JSON and vice versa. Supports duplicate keys as arrays. Essential for API debugging and URL parameter analysis.

CSV ↔ JSON Converter

Online CSV to JSON and JSON to CSV converter. Parses CSV with custom delimiters, handles quoted fields and escaped characters. Convert spreadsheet data to JSON arrays and back. Essential for data migration, API testing, and ETL pipelines.

JSON → TypeScript Interfaces

Automatically generate TypeScript interfaces from JSON data. Infers types for strings, numbers, booleans, arrays, and nested objects. Creates clean, exportable interface definitions. Essential for frontend developers working with APIs.

Text Diff — Compare Two Texts

Online text comparison tool using LCS (Longest Common Subsequence) algorithm. Shows added, removed, and unchanged lines with unified diff format. Compare any text, code, configs, or documents. Complements the JSON Diff tool for non-JSON content.
