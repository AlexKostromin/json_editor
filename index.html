DevTools Online — бесплатные инструменты разработчика

DevTools Online — набор из 12 бесплатных онлайн инструментов для веб-разработчиков, backend-инженеров, DevOps и QA специалистов. Все инструменты работают прямо в браузере, данные не отправляются на сервер. Можно установить как PWA-приложение для использования офлайн.

Онлайн JSON редактор

Бесплатный JSON Editor Online с подсветкой синтаксиса, форматированием, валидацией и древовидным просмотром. Поддерживает режимы tree, text и table для удобной работы с JSON данными любого размера. Скачивание и загрузка файлов, вставка из буфера обмена.

JWT Decoder — декодирование токенов

Онлайн JWT декодер: вставьте JSON Web Token и мгновенно получите расшифрованные header и payload. Автоматическое преобразование Unix timestamp полей (exp, iat, nbf) в читаемые даты. Безопасно — токен не покидает ваш браузер. Работает с Auth0, Firebase, Keycloak и любыми другими JWT.

Base64 Encode / Decode

Кодирование и декодирование Base64 онлайн. Поддержка кириллицы и Unicode. Конвертируйте текст в Base64 и обратно мгновенно в браузере. Используйте для работы с API, email-вложениями и конфигурационными файлами.

URL Encode / Decode

URL кодирование и декодирование онлайн. Преобразование специальных символов, кириллицы и пробелов в URL-safe формат и обратно. Необходимо для работы с query-параметрами, API запросами и веб-ссылками.

Unix Timestamp конвертер

Конвертер Unix Timestamp в человекочитаемую дату и обратно. Поддержка секунд и миллисекунд, отображение в UTC и локальном времени (MSK). Полезно для анализа логов, баз данных и API-ответов.

UUID v4 генератор

Генератор случайных UUID v4 онлайн. Создание уникальных идентификаторов для баз данных, API и тестирования. Генерирует 5 UUID за раз с использованием криптографически стойкого crypto.randomUUID().

YAML ↔ JSON конвертер

Конвертация YAML в JSON и обратно онлайн. Мгновенное преобразование конфигурационных файлов Kubernetes, Docker Compose, Ansible, GitHub Actions между форматами. Поддержка всех типов данных YAML.

JSON Schema генератор

Автоматическая генерация JSON Schema (draft 2020-12) из JSON данных. Определение типов, форматов (date-time, email, uri) и обязательных полей. Используйте для валидации API, OpenAPI спецификаций и конфигураций.

Hash генератор — SHA-256, SHA-1, SHA-512

Генерация хэшей SHA-256, SHA-1, SHA-384, SHA-512 онлайн. Хэширование текста через Web Crypto API прямо в браузере. Безопасно — данные не покидают ваш компьютер. Подходит для проверки целостности данных и контрольных сумм.

Regex Tester — тест регулярных выражений

Онлайн тестер регулярных выражений с поддержкой флагов g, i, m. Показывает все совпадения, группы захвата и позиции. Незаменимый инструмент для отладки regex в JavaScript, Python, Go и других языках.

Color Converter — HEX, RGB, HSL

Конвертер цветов между форматами HEX, RGB и HSL онлайн. Мгновенное преобразование цветовых значений для CSS, Tailwind, Material Design. Поддержка 3- и 6-значных HEX кодов.

JSON Diff — сравнение JSON

Сравнение двух JSON объектов онлайн. Глубокое рекурсивное сравнение с подсчётом добавленных, удалённых и изменённых полей. Показывает путь к каждому изменению. Полезно для отладки API и сравнения конфигураций.

Mock Data Generator — фейковые данные

Генератор фейковых данных для тестирования. Создание пользователей с русскими именами, email, телефонами, городами, компаниями и другими полями. До 100 записей за раз. Идеально для seed-данных, тестов и прототипирования.
